## 🔍 Overview

<a href="#betterDisplayPro">BetterDisplay Pro</a> serves Mac users with external display setups who have encountered macOS's display configuration limitations — the missing HiDPI modes, the brightness ceiling on MacBook Pro displays, the window collapse problem with headless Mac mini setups, and the inadequate per-display control that System Settings provides.

The <a href="#betterDisplayPro">external monitor power user use case</a> is the most common — Mac users with 4K, 5K, or ultrawide external monitors who find that System Settings offers insufficient resolution scaling options, excludes HiDPI modes the display is capable of rendering, or provides no control over the monitor's hardware brightness through software. BetterDisplay Pro addresses all three limitations. The <a href="#betterDisplayPro">Mac mini headless remote access use case</a> serves Mac mini users who run their machine without a physical display — for home server use, remote access workflows, or shared resource setups where a permanent display is unnecessary. Without virtual displays, macOS destroys the window arrangement every time the machine is accessed without a display; BetterDisplay Pro's virtual displays prevent this.

The <a href="#betterDisplayPro">MacBook Pro outdoor and bright environment use case</a> unlocks the XDR brightness capability built into the hardware but restricted by Apple's software — users who work outdoors, in vehicles, or in bright office environments where standard brightness is genuinely insufficient benefit from the additional brightness range. <a href="#betterDisplayPro">Multi-display professional setup use case</a> uses per-display color temperature, arrangement memory, and independent night mode scheduling to maintain consistent multi-monitor working environments across different lighting conditions and use contexts.

<a href="#betterDisplayPro">Refresh rate management</a> switches display refresh rates through the menu bar interface — particularly useful for ProMotion-capable displays and monitors supporting multiple refresh rates where switching between content modes benefits from different refresh rates. <a href="#betterDisplayPro">Custom ICC color profile support</a> creates, applies, and manages color profiles per display for color-critical photography, design, and print production work.

<a href="#betterDisplayPro">Setapp availability</a> includes BetterDisplay Pro through the Setapp subscription for subscribers who want access alongside other Mac utilities in a single subscription. <a href="#betterDisplayPro">Active development by waydabber</a> delivers regular updates maintaining compatibility with macOS releases and expanding capabilities as Mac display hardware evolves — a responsive developer with an engaged community of Mac power users.
